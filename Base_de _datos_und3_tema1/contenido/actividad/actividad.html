🖱️ Actividad 2: Arrastrar y Soltar - Parámetros de Conexión

Arrastra cada parámetro a su descripción correcta para aprender cómo configurar una conexión a base de datos.

host
port
user

database

🏠 host

Servidor donde corre la base de datos (localhost, 192.168.1.10)

🔢 port

Puerto de comunicación (3306 para MySQL, 5432 para PostgreSQL)

👤 user

Usuario con acceso a la base de datos (root, admin, app_user)

🔐 password

Contraseña del usuario para autenticarse

📦 database

Nombre de la base de datos a usar (tienda_db, usuarios_db)

📚 driver

Librería que traduce órdenes al lenguaje de la BD (mysql2, pg, psycopg2)

♻️ pool

Conjunto de conexiones reutilizables para mejor rendimiento

🔤 charset

Codificación de caracteres (utf8mb4, utf8)
