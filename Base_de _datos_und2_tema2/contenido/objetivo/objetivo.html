Objetivos de aprendizaje

  • 🧠 Ejecutar operaciones de inserción, consulta, actualización y eliminación de datos utilizando sintaxis SQL estándar.
  • 🔍 Validar la integridad de la información mediante el uso correcto de cláusulas de filtrado y condiciones.
  • 📝 Integrar lógica de manipulación de datos en escenarios prácticos que simulen el comportamiento de una base de datos en una aplicación real.
