Objetivos de aprendizaje

  • 🧠 Comprender la arquitectura REST y los principios que guían el diseño de endpoints para operaciones CRUD, identificando el método HTTP y el código de estado correspondiente a cada operación.
  • 🔍 Estructurar endpoints claros y semánticos que faciliten la comunicación entre el front-end y la base de datos.
  • 📝 Implementar la lógica de negocio necesaria para procesar solicitudes de datos y devolver respuestas en formato JSON.
