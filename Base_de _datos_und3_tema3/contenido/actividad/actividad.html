🖱️ Actividad 2: Arrastrar y Soltar - Clasifica Sentencias DDL

Arrastra cada sentencia SQL a la operación DDL que le corresponde (CREATE, ALTER o DROP).

CREATE TABLE usuarios (...)
ALTER TABLE clientes ADD telefono VARCHAR(15)
DROP DATABASE sistema_ventas
CREATE INDEX idx_nombre ON empleados(nombre)
ALTER TABLE productos MODIFY COLUMN precio DECIMAL(10,2)
DROP TABLE temp_logs
CREATE VIEW vista_activos AS SELECT * FROM usuarios WHERE activo=1
ALTER TABLE pedidos DROP COLUMN observaciones

📌 CREATE

Crear nuevos objetos (tablas, índices, vistas)

✏️ ALTER

Modificar estructura existente

🗑️ DROP

Eliminar objetos permanentemente
